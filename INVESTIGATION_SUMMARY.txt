================================================================================
THE PROBLEM
================================================================================

Users expect to see multi-agent discussions in the "决策" sidebar when asking
strategy-related questions. Instead, the sidebar remains empty.

Why? Two separate agent systems exist with ZERO integration:
  1. Arena System      ✅ Can run multi-agent discussions (debate/collab/review)
  2. Chat Orchestrator ✅ Routes messages to agents
  3. Connection        ❌ MISSING

================================================================================
ROOT CAUSES (5 Total)
================================================================================

1. NO ARENA TRIGGER FROM CHAT (CRITICAL)
   ├─ Chat router never creates or invokes arena discussions
   ├─ File: src/stock_datasource/modules/chat/router.py (line 196-362)
   ├─ Evidence: No imports of arena modules, no arena API calls
   └─ Impact: "决策" sidebar data has nowhere to come from

2. INCOMPATIBLE EVENT MODELS (HIGH)
   ├─ Chat emits: debug events (classification, routing, tool_result)
   ├─ Arena emits: thinking messages (argument, conclusion, debate)
   ├─ File: Both systems use different event structures
   └─ Impact: Frontend can't display both in same sidebar

3. HANDOFF CONFIG UNUSED (HIGH)
   ├─ AGENT_HANDOFF_MAP defined in execution_planner.py (line 62-68)
   ├─ Maps agent routing targets but never consulted
   ├─ File: src/stock_datasource/services/execution_planner.py
   └─ Impact: A2A communication incomplete

4. DEBUG EVENTS INCOMPLETE (MEDIUM)
   ├─ Handoff debug event type defined but never emitted
   ├─ File: src/stock_datasource/agents/orchestrator.py
   ├─ Evidence: Frontend expects it, backend never sends it
   └─ Impact: Routing discussions invisible to users

5. FRONTEND MISSING ROLE (LOW)
   ├─ DebugMessage interface missing "discussion" role
   ├─ File: frontend/src/stores/chat.ts (line 75-88)
   ├─ Has: orchestrator, agent, tool, system, handoff
   ├─ Needs: discussion (for arena events)
   └─ Impact: Can't display discussion events even if sent
